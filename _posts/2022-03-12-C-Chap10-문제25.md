> **조건**

- 함수는 순환함수로 작성한다.
- 함수의 이름은 gugudan이고, 매개변수는 1개의 정수이다.
- 함수의 매개변수는, 2단의 범위를 지정하는 숫자이다.

> **Source**

```C++
#include<stdio.h>

void gugudan(int i) {
	if (i >= 1) {
		printf("2 * %d = %d\n", i, i * 2);
		gugudan(i - 1);
	}
	else
		printf("###END###");
}

int main(void) {
	int range = 0;

	printf("2단의 범위를 입력하시오: ");
	scanf("%d", &range);

	gugudan(range);

	return 0;
}
```
